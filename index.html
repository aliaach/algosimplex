- Simplex Solver -

L'algorithme du simplexe est une méthode couramment utilisée pour résoudre les problèmes de programmation linéaire. Il s'agit d'une procédure itérative qui part d'une solution faisable initiale et se déplace systématiquement d'une solution faisable à une autre, améliorant la valeur de la fonction objectif à chaque étape jusqu'à ce qu'une solution optimale soit atteinte.

Voici un bref aperçu de son fonctionnement :

  1. Initialisation : Commencez par une solution faisable initiale. Cela pourrait être n'importe quelle solution faisable, comme l'origine (0,0) si toutes les variables sont non négatives.
  2. Identification du pivot : Sélectionnez un élément pivot dans la solution actuelle. L'élément pivot est choisi en fonction du coefficient le plus négatif dans la ligne de la fonction objectif (si on maximise) ou le coefficient le plus positif (si on minimise).
  3. Opération pivot : Effectuez des opérations pivot pour mettre à jour la solution. Cela implique de sélectionner une ligne pivot et une colonne pivot, et d'utiliser l'élimination de Gauss pour transformer l'élément pivot en 1 et rendre tous les autres éléments de la colonne pivot à zéro.
  4. Test d'optimalité : Vérifiez si la solution actuelle est optimale. Si tous les coefficients dans la ligne de la fonction objectif sont non négatifs (pour la maximisation) ou non positifs (pour la minimisation), alors la solution est optimale. Sinon, répétez les étapes 2 à 4.
  5. Itération : Si la solution n'est pas optimale, revenez à l'étape 2 et continuez à itérer jusqu'à ce qu'une solution optimale soit trouvée.

Le problème de programmation linéaire que vous devez saisir ci-dessous comprendrait généralement :

  • Fonction Objectif : C'est la fonction que vous voulez maximiser ou minimiser.
  • Contraintes : Ce sont les conditions qui doivent être satisfaites par les variables de décision.
  • Variables de Décision : Ce sont les variables que vous pouvez ajuster pour optimiser la fonction objectif.

Une fois que vous fournirez le problème de programmation linéaire, l'algorithme du simplexe pourra être appliqué pour trouver la solution optimale. Si vous souhaitez un exemple pour voir comment configurer un problème de programmation linéaire, vous pouvez appuyer sur le bouton "Exemple".
